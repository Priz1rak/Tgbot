Задумка бота - показать время намаза по всей России
При вводе команды /start бот приветствует пользователя и предлагает выбор команд
"Время намаза" - Предлагает выбрать город из списка(пока не работает) либо указать город с помощью клавиатуры
Выбрав кнопку "Местонахождения Киблы" бот переходит на веб сайт https://qiblafinder.withgoogle.com/intl/ru/desktop/finder
нажав на кнопку 'научиться совершать намаз' бот отправляет видео формата mp4
При каждом запросе в файл requests.txt записывается точная дата, а также город из запроса. Можно в дальнейшем использовать для собирания каких то статистик.
 1. requirements.txt - есть
2. ORM модели - нет
3. работа с контекстом пользователя - есть (отправка местоположения)
загрузка и использование файлов: картинки музыка - есть(отправление иконок погоды)
4. Использование стороннего API - есть (openweathermap, Yandex геокодер)
5. Хранение данных txt/csv - есть (запись данных запросов в файл requests.txt)
6. Хостинг - есть, реализовывал на сайте pythonanywhere.com
